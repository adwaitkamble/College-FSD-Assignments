# Assignment 2 Portfolio Website

This project is a personal portfolio website built for Assignment 2 using HTML, CSS, and JavaScript. It presents profile information, education, experience, projects, technical skills, certifications, and contact details in a responsive single-page layout.

## Project Overview

The website is designed as a modern portfolio for Adwait Kamble. It uses a clean glassmorphism-inspired UI, section-based navigation, smooth scrolling, scroll reveal effects, and interactive contact actions.

## Features

- Responsive single-page portfolio layout
- Sticky navigation bar with smooth section scrolling
- Hero section with social and contact links
- About, Education, Experience, Projects, Skills, and Certifications sections
- Contact section with phone, email, GitHub, and LinkedIn links
- Scroll-based reveal animations
- Cursor glow visual effect
- Gmail or mail client launch for email CTA buttons

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Font Awesome icons

## How to Run

1. Open the project folder.
2. Open `assignment_2.html` in any web browser.

You can also use VS Code Live Server for a better development preview.

## Main Files

- `assignment_2.html`: Main structure and content of the portfolio website
- `assignment_2.css`: Styling, layout, responsiveness, and animations
- `script.js`: Interactive behavior such as smooth scrolling, reveal effects, header state, and email actions
- `adwait photo.jpg`: Profile image used in the hero section
- `code+output.docx`: Assignment documentation/output file
